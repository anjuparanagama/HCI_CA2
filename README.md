# 🌐 Personal Portfolio Website

A modern and responsive personal portfolio website built with **React.js**, designed to showcase my skills, projects, and background as a web developer. It serves as an online resume and project gallery for potential employers, clients, or collaborators.

## 🚀 Live Demo

🌍 [View Live Website]([https://your-portfolio-link.com](https://hci-ca-2-anjuparanagamas-projects.vercel.app/))

---

## 🛠️ Technologies Used

| Technology        | Purpose                                          |
|-------------------|--------------------------------------------------|
| React.js          | Frontend framework for building UI               |
| React Router DOM  | For smooth client-side routing/navigation        |
| Tailwind CSS      | Utility-first CSS for fast and responsive design |
| Vite              | Lightweight, fast build tool and dev server      |
| ESLint            | Code quality and formatting                      |
